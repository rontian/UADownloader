# UA Downloader - 快速启动指南

## 准备工作

### 1. 检查依赖

确保以下组件已安装：
- Unity Editor (2019.4+)
- AssetInventory 插件
- Newtonsoft.Json (通常Unity已包含)

### 2. 验证安装

打开Unity项目后，检查：
1. Console窗口是否有编译错误
2. 菜单栏是否出现：`Tools → UA Downloader → Package Batch Downloader`

## 首次使用

### 步骤 1：登录Asset Store

1. 在Unity编辑器中打开：`Window → Asset Store`
2. 点击右上角的用户图标登录
3. 使用你的Unity ID登录

### 步骤 2：打开下载器

1. 点击菜单：`Tools → UA Downloader → Package Batch Downloader`
2. 窗口会打开，显示批量下载器界面

### 步骤 3：设置导出目录

1. 点击"浏览"按钮
2. 选择一个用于存放下载文件的目录
3. 确保该目录有足够的磁盘空间

### 步骤 4：获取资源列表

1. 点击"获取资源列表快照"按钮
2. 等待几秒到几分钟（取决于你购买的资源数量）
3. 完成后会显示：`成功获取 N 个资源包`

### 步骤 5：开始下载

1. 点击"开始批量下载"按钮
2. 下载会自动开始，你可以在列表中看到进度
3. 已完成的资源会显示绿色 ✓，失败的显示红色 ✗

### 步骤 6：等待完成

- 下载可能需要较长时间，取决于资源数量和大小
- 你可以随时暂停或取消下载
- 下载完成的文件在导出目录中，格式为：`资源包名_版本.unitypackage`

## 常用操作

### 暂停下载

1. 点击"暂停"按钮
2. 当前下载会停止，进度自动保存
3. 下次点击"开始批量下载"会从上次停止的位置继续

### 取消下载

1. 点击"取消"按钮
2. 下载停止，进度文件被删除
3. 下次开始会从头开始下载

### 查看下载结果

1. 打开你设置的导出目录
2. 查看 `packages.json`（资源列表）
3. 查看 `index.json`（下载进度）
4. 查看 `.unitypackage` 文件（下载完成的资源）

## 故障排除

### 问题1：菜单中没有"UA Downloader"选项

**可能原因**：
- 脚本未编译成功
- Assembly Definition引用错误

**解决方法**：
1. 检查Unity Console是否有编译错误
2. 尝试：`Assets → Reimport All`
3. 重启Unity编辑器

### 问题2：提示"未登录Asset Store"

**解决方法**：
1. 打开：`Window → Asset Store`
2. 确保已登录（右上角显示你的用户名）
3. 如果未登录，点击登录按钮

### 问题3：获取列表返回空

**可能原因**：
- 网络连接问题
- 没有购买过任何资源
- Token过期

**解决方法**：
1. 检查网络连接
2. 重新登录Asset Store
3. 查看Console是否有错误信息

### 问题4：下载一直卡住

**解决方法**：
1. 点击"暂停"，等待几秒
2. 点击"开始批量下载"重新开始
3. 检查Unity Console的日志
4. 如果问题持续，尝试重启Unity编辑器

### 问题5：文件复制失败

**解决方法**：
1. 检查导出目录的写入权限
2. 检查磁盘空间是否充足
3. 确保导出目录路径不包含特殊字符

## 性能优化建议

### 磁盘空间

- Unity缓存目录需要空间（所有资源大小总和）
- 导出目录需要相同大小的空间
- 建议至少预留 2倍 的资源总大小空间

### 网络带宽

- 下载速度取决于你的网络连接
- 建议在网络稳定时进行批量下载
- 避免在高峰时段下载

### 下载策略

- 可以先获取快照，查看需要下载的资源数量和大小
- 可以暂停下载，分批次完成
- 失败的下载可以单独重试

## 高级用法

### 手动编辑 packages.json

如果你只想下载部分资源：
1. 获取完整快照后，打开 `packages.json`
2. 删除不需要的资源条目
3. 保存文件
4. 开始下载（只会下载列表中的资源）

### 恢复中断的下载

如果Unity编辑器意外关闭：
1. 重新打开Unity和下载器窗口
2. 设置相同的导出目录
3. 工具会自动加载 `index.json`
4. 点击"开始批量下载"继续

### 批量导入

下载完成后，如果需要导入到项目：
1. 在Unity中：`Assets → Import Package → Custom Package`
2. 选择下载的 `.unitypackage` 文件
3. 按提示完成导入

## 文件说明

### packages.json
包含所有购买的资源包信息，可以用于：
- 查看购买历史
- 统计资源数量
- 过滤需要下载的资源

### index.json
记录下载进度，包含：
- 当前下载位置
- 已完成的资源ID列表
- 失败的资源ID列表
- 最后更新时间

### *.unitypackage
实际的资源包文件，可以：
- 导入到Unity项目
- 备份到其他位置
- 分享给团队成员（注意许可协议）
